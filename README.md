# 俳句と絵文字
![](https://naoyashiga.github.io/HaikuWithEmoji/ogp.png)
俳句に絵文字を無作為に付加することで、俳句の持つコンテキストを崩壊させる。絵文字はプラットフォームの仕様によりデザインが変わる。絵文字を使う以上、俳句が表現する世界はプラットフォームに依存する。

# Haiku with Emoji
Adding a emoji to haiku makes the context change. The design of emoji depends on the platform. What haiku with emoji express depends on the platform.
